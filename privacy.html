隐私说明

你的健康数据，你做主

数据存储

  • 你的健康数据（BMI 记录、月经周期、症状等）存储在 Supabase 云端数据库中，采用加密传输（HTTPS）。
  • 数据同步采用离线优先策略：本地缓存 + 云端同步，确保断网时也能正常使用。
  • API 密钥仅在内存中临时保存，不会持久化到浏览器本地存储。

数据安全

  • 所有 API 请求均通过 JWT 令牌认证，确保只有你本人能访问自己的数据。
  • 服务端通过 user_id 隔离数据，不同用户之间完全隔离。
  • 我们不会将你的健康数据分享给任何第三方。

你的权利

  • 导出数据：你可以在"历史记录"页面导出所有数据（CSV 或 JSON 格式）。
  • 删除数据：你可以在"历史记录"页面清空所有记录，或联系管理员删除服务端数据。
  • 注销账号：如需完全删除账号及所有数据，请联系我们。

免责声明

  • 本应用提供的健康分析、周期预测和饮食建议仅供参考，不构成医学诊断或治疗建议。
  • 易孕窗口和排卵预测基于统计估算，不能作为避孕或备孕的依据。
  • 如有健康问题，请咨询专业医疗机构。
